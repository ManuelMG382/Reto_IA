# Predicción de Tiempos de Entrega (ETA Prediction)

## Objetivo del proyecto
El objetivo de este proyecto es predecir el tiempo estimado de entrega (ETA) en un contexto de delivery/logística utilizando modelos de Machine Learning.
Se exploran dos enfoques distintos:
- **Regresión**: predicción continua del tiempo de entrega en minutos.
- **Clasificación**: predicción directa del rango temporal de entrega.
El proyecto analiza el impacto del *feature engineering* sobre ambos enfoques y compara su rendimiento utilizando modelos basados en XGBoost.
